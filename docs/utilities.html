Place elements inline

Place elements inline. Useful for navs, horizontal lists, controls etc.

Home
About
Pricing
Contact
<div class="u-inline">
  <div>Home</div>
  <div>About</div>
  <div>Pricing</div>
  <div>Contact</div>
</div>

Truncate text

Truncate a span of text to the width of its parent element and append 3 dots at the point of truncation. This only works with block level elements.

This is an unnecessarily long sentence to demonstrate the truncation of text.

<p class="u-truncateText">This is an unnecessarily long sentence to demonstrate the truncation of text.</p>

Center horizontally

Horizontally center an element within its parent element. This is useful for page wrappers and other similar use cases.

This element is centered horizontally.
<div class="u-centered"></div>

Center vertically

Vertically center an element within its parent element. The parent element must have a height explicitly set, but the centered element can have a variable height.

This text is centered vertically.
<div style="height: 200px;">
        <div class="u-verticallyCenter">This text is centered vertically.</div>
      </div>

Clear floats

A clearfix for clearing floats. When an element is floated, its parent element will lose its height unless a height is explicitly defined. This clearfix class will extend the element's boundary to the height of its children.

This element's floats are cleared.
<div class="u-cf">
        <div class="float-left">This element's floats are cleared.</div>
      </div>

Visually hide elements

Hide an element from the browser but keep it accessible to screen-readers.

This element is hidden.
<div class="u-visuallyHidden">This element is hidden.</div>

Hide elements

Hide an element from both screen-readers and browsers.

This element is hidden.
<div class="u-hidden">This element is hidden.</div>

Positioning elements

Pin elements to corners or stretch elements across the entire viewport. Useful for modal screens etc.

<div class="u-pinTL">This element is pinned to the top left edge of its closest relative parent.</div>
      <div class="u-fixTL">This element is fixed to the top left edge of its closest relative parent.</div>
      <div class="u-pinAll">This element is pinned to all corners of the viewport.</div>
      <div class="u-fixAll">This element is fixed to all corners of the viewport.</div>
